# Структура изображений для Recloth

## Необходимые изображения:

### Корневая папка images/
- `logo.png` - Логотип Recloth (280x90px)
- `hero-bg.jpg` - Фоновое изображение для Hero секции (1920x1080px или больше)
- `icon-192.png` - Иконка для PWA (192x192px)
- `icon-512.png` - Иконка для PWA (512x512px)

### images/articles/
Статьи (рекомендуемый размер: 600x400px):
1. `jeans-ideas.jpg` - 10 идей для старых джинсов
2. `t-shirt-bag.jpg` - Сумка из футболки
3. `sweater-trends.jpg` - Апсайклинг свитеров
4. `eco-materials.jpg` - Эко-материалы
5. `mistakes.jpg` - Ошибки новичков
6. `interview.jpg` - Интервью с мастерами

### images/ideas/
Идеи трансформаций (рекомендуемый размер: 400x400px):
**ДО:**
- `shirt-before.jpg` - Рубашка до
- `tshirt-before.jpg` - Футболка до
- `rug-before.jpg` - Коврик до
- `jeans-before.jpg` - Джинсы до
- `sweater-before.jpg` - Свитер до
- `jacket-before.jpg` - Куртка до

**ПОСЛЕ:**
- `shirt-after.jpg` - Платье после
- `tshirt-after.jpg` - Сумка после
- `rug-after.jpg` - Чехол после
- `jeans-after.jpg` - Юбка после
- `sweater-after.jpg` - Пончо после
- `jacket-after.jpg` - Рюкзак после

### images/gallery/
Галерея работ пользователей (рекомендуемый размер: 600x600px или больше):
- `gallery-1.jpg` до `gallery-12.jpg` (минимум 12 изображений)

## Форматы:
- Рекомендуется использовать JPG для фотографий
- PNG для логотипов и иконок
- Оптимизируйте изображения для веб (используйте инструменты типа TinyPNG)

## Placeholder изображения:
Если у вас пока нет изображений, можно использовать placeholder сервисы:
- https://via.placeholder.com/600x400
- https://picsum.photos/600/400

Или создать простые цветные изображения с текстом для тестирования.
